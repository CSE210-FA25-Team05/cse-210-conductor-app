(constant) authService

Auth Routes - GET /auth/oauth/google → Redirect to Google with CSRF state cookie - GET /auth/oauth/google/callback → Server-side exchange, create session, set "sid" cookie - POST /auth/oauth/google/add_token → Optional SPA/PKCE flow, same behavior as callback - POST /auth/logout → Delete session + clear session cookie - GET /me/profile → Get current user's profile - POST /me/profile → Update current user's profile and mark it complete
